# Prompts

This project is built for fast, honest MVP delivery.

## Product messaging
- Focus on AI spend waste, not vague AI buzzwords
- Use concrete savings and recommendations
- Highlight the audit workflow: enter tools, review recommendations, share results

## Copy principles
- Clear: simple inputs, concise results
- Honest: optional persistence, fallback when features are not configured
- Practical: convey cost savings and actionable next steps
